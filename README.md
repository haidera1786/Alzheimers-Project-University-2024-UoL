# Alzheimer's MRI Classification Using CNN and ResNet50

## Overview
This project explores deep learning techniques for detecting stages of Alzheimer's disease using MRI images.

Models implemented:
- Custom CNN
- Transfer Learning with ResNet50

## Dataset
Alzheimer's MRI dataset with four classes:
- Mild Dementia
- Moderate Dementia
- Non Dementia
- Very Mild Dementia

## Models
1. Custom CNN
2. ResNet50 Transfer Learning

## Results
Accuracy, confusion matrix, and ROC analysis were used to evaluate performance.
